**Header block** (Phase 69 DOC-01 format):
```
# Purpose: Auto-generate docs/REFERENCE_MANUAL.md by parsing structured 5-field headers from all pipeline scripts. Builds dependency matrix and writes onboarding documentation.
# Inputs: R/*.R and R/utils/*.R script files (header blocks)
# Outputs: docs/REFERENCE_MANUAL.md (complete reference manual)
# Dependencies: (standalone generator script), glue, stringr
# Requirements: DOC-04
```

**SECTION 1: SETUP**:
- `rm(list = ls())`
- `library(glue)`, `library(stringr)`
- Define output path: `output_path <- "docs/REFERENCE_MANUAL.md"`

**SECTION 2: HEADER PARSING FUNCTION**:
Create `parse_script_header(filepath)` function that:
1. Reads file with `readLines(filepath, warn = FALSE)`
2. Extracts the header block (lines between first `# ==` and the closing `# ==` or first non-comment line)
3. For each of the 5 fields (Purpose, Inputs, Outputs, Dependencies, Requirements):
   - Find the line matching `^#\\s*(Purpose|Inputs|Outputs|Dependencies|Requirements):\\s*(.*)$`
   - Extract the first-line value after the colon
   - For multi-line fields: collect subsequent comment lines (matching `^#\\s+` with leading whitespace) until the next field keyword or `# ==` boundary
   - Join multi-line values with `, ` separator (flatten to single line for table)
4. Return a named list: `list(script = basename(filepath), purpose = ..., inputs = ..., outputs = ..., dependencies = ..., requirements = ...)`
5. If a field is not found, return "Not documented"

**SECTION 3: CONFIG CONSTANT DETECTION**:
Create `detect_config_constants(filepath)` function that:
1. Reads file with `readLines(filepath, warn = FALSE)`
2. Defines the 11 known constants: `c("CONFIG", "EXTRACT_DATE", "PCORNET_TABLES", "PCORNET_PATHS", "ICD_CODES", "PAYER_MAPPING", "AMC_PAYER_LOOKUP", "TREATMENT_CODES", "ANALYSIS_PARAMS", "CANCER_SITE_MAP", "TIER_MAPPING")`
3. For each constant, check if it appears in any non-comment line (not starting with `#`)
4. Return comma-separated string of used constants, or "None" if none found
5. Exclude R/00_config.R itself from detection (it defines them, doesn't use them as a consumer)

**SECTION 4: PARSE ALL SCRIPTS**:
1. Get numbered scripts: `list.files("R", pattern = "^[0-9]+.*\\.R$", full.names = TRUE)` sorted by filename
2. Get utils scripts: `list.files("R/utils", pattern = "\\.R$", full.names = TRUE)` sorted by filename
3. Parse headers for all numbered scripts into a list of named lists
4. Parse headers for all utils scripts into a separate list
5. Detect config constant usage for each script
6. Print progress: `message(glue("Parsed {length(numbered_headers)} numbered scripts + {length(utils_headers)} utils modules"))`

**SECTION 5: GENERATE MARKDOWN**:
