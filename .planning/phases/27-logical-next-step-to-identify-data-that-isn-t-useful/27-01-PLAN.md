**SECTION 3: QA Dimension Functions**

Implement these four functions:

**3a. completeness_profile(df, table_name)** (per D-04):
- For each column in df, compute: n_total (nrow), n_non_na (sum(!is.na())), pct_complete (round(100 * n_non_na / n_total, 1)), n_distinct (n_distinct(na.rm=TRUE)), n_sentinel (sum of values matching is_sentinel(), only for character columns)
- Use summarise(across(everything(), ...)) with pivot_longer to stack results
- Add table column, arrange by pct_complete ascending (worst completeness first)
- Return a tibble with columns: table, column, n_total, n_non_na, pct_complete, n_distinct, n_sentinel, pct_sentinel (for character cols; 0 for non-character)

**3b. validity_check(df, table_name, value_sets)** (per D-05):
- Find intersection of names(df) and names(value_sets) to identify coded columns
- For each coded column, filter rows where value is not NA and not in valid set
- Return a tibble with columns: table, column, invalid_value, n_invalid, valid_values_sample (first 5 valid values comma-separated for reference)
- If no invalid values found for any column, return a zero-row tibble with same columns

**3c. duplicate_scan(df, table_name, unique_id_col = NULL)** (per D-06):
- Tier 1: Exact row duplicates on ALL columns using duplicated(df) | duplicated(df, fromLast = TRUE)
- Tier 2: If unique_id_col is not NULL, also check "semantic duplicates" on all columns EXCEPT unique_id_col using duplicated(df_no_id) | duplicated(df_no_id, fromLast = TRUE)
- For each tier, group identical rows and count n_copies
- Return a tibble with columns: table, tier (character: "exact" or "semantic"), total_duplicate_rows, unique_duplicate_sets, max_copies
- If no duplicates, return rows with 0 values

**3d. overlap_detect(df, table_name, date_col, id_col = "ID")** (per D-03, D-07):
- Guard clause: if date_col is NULL or !"SOURCE" %in% names(df), return tibble(table=table_name, overlap_groups=0, patients_affected=0, top_combo=NA_character_, skip_reason="...")
- Ensure date column is Date type (it should be from load_pcornet_table, but guard with as.Date() if character)
- Group by id_col + date column, count n_sources = n_distinct(SOURCE), build source_combo = paste(sort(unique(SOURCE)), collapse="+")
- Filter groups where n_sources > 1
- Return a tibble with: table, overlap_groups (nrow of filtered), patients_affected (n_distinct of id_col), top_combo (most frequent source combination), pct_of_total_dates (overlap_groups / total unique id+date pairs * 100, rounded to 2 decimal places)

```r
qa_results <- list()
for (tbl_name in qa_tables) {
  df <- pcornet[[tbl_name]]
  if (is.null(df)) { message(glue("  SKIP: {tbl_name} not loaded")); next }

  message(glue("\n--- {tbl_name} ({format(nrow(df), big.mark=',')} rows, {ncol(df)} cols) ---"))

  # Completeness
  qa_results[[tbl_name]]$completeness <- completeness_profile(df, tbl_name)

  # Validity
  qa_results[[tbl_name]]$validity <- validity_check(df, tbl_name, VALUE_SETS)

  # Exact + semantic duplicates
  uid_col <- TABLE_UNIQUE_ID_COLS[[tbl_name]]
  qa_results[[tbl_name]]$duplicates <- duplicate_scan(df, tbl_name, uid_col)

  # Multi-source overlap
  date_col <- TABLE_DATE_FIELDS[[tbl_name]]
  id_col <- TABLE_ID_FIELDS[[tbl_name]]
  qa_results[[tbl_name]]$overlap <- overlap_detect(df, tbl_name, date_col, id_col)

  # Log key findings inline
  comp <- qa_results[[tbl_name]]$completeness
  low_cols <- comp %>% filter(pct_complete < 50) %>% nrow()
  if (low_cols > 0) message(glue("  Completeness: {low_cols} columns below 50% complete"))

  n_invalid_total <- sum(qa_results[[tbl_name]]$validity$n_invalid, na.rm = TRUE)
  if (n_invalid_total > 0) message(glue("  Validity: {format(n_invalid_total, big.mark=',')} invalid values across {nrow(qa_results[[tbl_name]]$validity)} columns"))

  dup <- qa_results[[tbl_name]]$duplicates
  exact_dup <- dup %>% filter(tier == "exact") %>% pull(total_duplicate_rows) %>% sum()
  if (exact_dup > 0) message(glue("  Duplicates: {format(exact_dup, big.mark=',')} exact duplicate rows"))

  ovl <- qa_results[[tbl_name]]$overlap
  if (!is.na(ovl$overlap_groups) && ovl$overlap_groups > 0) {
    message(glue("  Overlap: {format(ovl$overlap_groups, big.mark=',')} multi-source groups ({ovl$top_combo})"))
  }
}
```
