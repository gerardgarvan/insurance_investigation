**Add to SECTION 3 (QA Dimension Functions):**

**3e. temporal_consistency(df, table_name, date_pairs):**
- For each date pair (start_col, end_col) defined for this table:
  - Count rows where start > end (logic violation)
  - Count rows where either date is in the future (> Sys.Date())
  - Count rows where either date has year == 1900 or year == 9999 (sentinel dates)
  - Count rows where either date has year < 1901 (SAS/Excel epoch sentinels)
- Also check ALL Date-type columns in df individually for future dates and sentinel years
- Return a tibble with columns: table, check_type (one of "start_after_end", "future_date", "sentinel_1900", "sentinel_9999", "pre_1901"), date_column (or "start_col / end_col" for pairs), n_violations, pct_violations (n_violations / nrow(df) * 100, rounded to 2 decimal places)
- If no date columns exist, return zero-row tibble with same schema

**3f. referential_integrity(df, table_name, ref_check, pcornet_list):**
- ref_check is the entry from REFERENTIAL_CHECKS for this table (or NULL if not defined)
- If NULL, return tibble(table=table_name, check="N/A", n_orphan_rows=0, n_orphan_ids=0, pct_orphan_rows=0, skip_reason="No FK defined")
- If ref_check defined:
  - Get parent table from pcornet_list: parent_df <- pcornet_list[[ref_check$parent_table]]
  - If parent_df is NULL, return with skip_reason = "Parent table not loaded"
  - Use anti_join: orphans <- df %>% anti_join(parent_df, by = setNames(ref_check$parent_col, ref_check$child_col))
  - Count n_orphan_rows = nrow(orphans), n_orphan_ids = n_distinct(orphans[[ref_check$child_col]])
  - Return tibble: table, check (glue("{child_col} -> {parent_table}.{parent_col}")), n_orphan_rows, n_orphan_ids, pct_orphan_rows (n_orphan_rows / nrow(df) * 100, rounded to 2)

**Update SECTION 4 (Run QA loop):**

Inside the per-table for loop, after the existing 4 dimensions, add:

```r
  # Temporal consistency
  date_pairs <- TABLE_DATE_PAIRS[[tbl_name]]
  qa_results[[tbl_name]]$temporal <- temporal_consistency(df, tbl_name, date_pairs)

  # Referential integrity
  ref_check <- REFERENTIAL_CHECKS[[tbl_name]]
  qa_results[[tbl_name]]$referential <- referential_integrity(df, tbl_name, ref_check, pcornet)

  # Log temporal findings inline
  temp <- qa_results[[tbl_name]]$temporal
  n_temporal_issues <- sum(temp$n_violations, na.rm = TRUE)
  if (n_temporal_issues > 0) message(glue("  Temporal: {format(n_temporal_issues, big.mark=',')} date logic violations"))

  # Log referential findings inline
  ref <- qa_results[[tbl_name]]$referential
  if (!is.na(ref$n_orphan_rows) && ref$n_orphan_rows > 0) {
    message(glue("  Referential: {format(ref$n_orphan_rows, big.mark=',')} orphan rows ({ref$n_orphan_ids} unique IDs)"))
  }
```
