<objective>
Create R/50_cancer_site_confirmation.R — a standalone analysis script that validates cancer site diagnosis codes by requiring at least 2 distinct dates per code per patient before counting the code as "confirmed." Produces a styled two-sheet xlsx comparing total vs confirmed patient counts at two matching levels: exact ICD-10 code and 3-character prefix.

Purpose: Epidemiological validation to distinguish real diagnoses (appearing on multiple dates) from potential rule-outs or data entry errors (single-date occurrences). This is the base for Phase 4's stricter 7-day separation requirement.

Output: R/50_cancer_site_confirmation.R script + cancer_site_confirmation.xlsx in output/tables/
</objective>

**Section 2 -- Load and classify DIAGNOSIS data:**
```r
dx_icd10 <- get_pcornet_table("DIAGNOSIS") %>%
  filter(DX_TYPE == "10") %>%
  select(ID, DX, DX_DATE) %>%   # DX_DATE added per D-03
  collect()

# Normalize codes (remove dots, uppercase)
dx_icd10 <- dx_icd10 %>%
  mutate(DX_norm = toupper(str_remove_all(DX, "\\.")))

# Filter to neoplasm codes (C00-D49)
dx_cancer <- dx_icd10 %>%
  filter(str_detect(DX_norm, "^[CD]"))

# Classify by prefix
dx_cancer <- dx_cancer %>%
  mutate(category = classify_codes(DX_norm))

# Handle unclassified codes (label for visibility, same as R/47)
n_unclassified <- sum(is.na(dx_cancer$category))
if (n_unclassified > 0) {
  unclass_codes <- dx_cancer %>% filter(is.na(category)) %>% pull(DX_norm) %>% unique()
  message(glue("  WARNING: {n_unclassified} rows ({length(unclass_codes)} unique codes) unclassified"))
  dx_cancer <- dx_cancer %>%
    mutate(category = ifelse(is.na(category), "Unclassified", category))
}
```

Log row counts at each step with `message()` and `glue()` using `format(x, big.mark=',')`.

confirmed_exact <- dx_cancer %>%
