Create two lookup functions using httr2:

`lookup_rxcui_name(rxcui, sleep_sec = 0.1)`:
- URL: `glue("https://rxnav.nlm.nih.gov/REST/rxcui/{rxcui}/properties.json")`
- Use `request(url) %>% req_timeout(10) %>% req_retry(max_tries = 3, is_transient = ~ resp_status(.x) %in% c(429, 503, 504)) %>% req_perform()`

  - If NULL or NA, return tibble with drug_name = NA, status = "ndc_not_found"
- Step 2: RxCUI -> Name via `lookup_rxcui_name()` (reuse the function above)
  - Sys.sleep(sleep_sec) between step 1 and step 2
- Return tibble(code = ndc, drug_name, lookup_status)

`lookup_drug_codes_batch(codes_df)`:
- Takes the extracted tibble, deduplicates by unique code + code_type combinations
- For RXNORM codes: call lookup_rxcui_name() in a for loop with progress logging every 10 codes
- For NDC codes: call lookup_ndc_to_name() in a for loop with progress logging every 10 codes
- bind_rows() all results
- Return tibble(code, drug_name, lookup_status)
- Log summary: N success, N not_found, N errors
