```
#   D-02: Detail table preserves one-row-per-code granularity
#   D-03: Separate rows by treatment type (concurrent as separate rows)
#   D-04: Full cohort (no filtering)
#   D-05: No payer tier data
#   D-06: CSV output only
#   D-07: Load from existing RDS artifacts
#
# Inputs:
#   - cache/outputs/treatment_episodes.rds (episode-level)
#   - cache/outputs/treatment_episode_detail.rds (detail-level)
#
# Outputs:
#   - output/gantt_episodes.csv (bars: one row per patient/type/episode)
#   - output/gantt_detail.csv (ticks: one row per patient/date/code)
# =============================================================================
```

**SECTION 1: SETUP AND CONFIGURATION**
- `suppressPackageStartupMessages({ library(dplyr); library(glue) })`

  - `DETAIL_RDS <- file.path(CONFIG$cache$outputs_dir, "treatment_episode_detail.rds")`
- Define output paths:
  - `OUTPUT_EPISODES <- file.path(CONFIG$output_dir, "gantt_episodes.csv")`
  - `OUTPUT_DETAIL <- file.path(CONFIG$output_dir, "gantt_detail.csv")`

**SECTION 2: LOAD INPUT DATA**
- Print banner: `message("=== Phase 01: Gantt Chart Data Export ===\n")`
- Check both RDS files exist with `file.exists()`. If either missing, call `stop(glue("ERROR: {path} not found. Run R/44_treatment_episodes.R first."))`.
- Load episodes: `episodes <- readRDS(EPISODES_RDS)` with `message(glue("  Loaded {nrow(episodes)} episode rows"))`
- Load detail: `detail <- readRDS(DETAIL_RDS)` with `message(glue("  Loaded {nrow(detail)} detail rows"))`

**SECTION 3: VALIDATE COLUMN STRUCTURE**
- Define expected episode columns: `c("patient_id", "treatment_type", "episode_number", "episode_start", "episode_stop", "episode_length_days", "distinct_dates_in_episode", "historical_flag", "triggering_codes")`
- Define expected detail columns: `c("patient_id", "treatment_type", "treatment_date", "triggering_code", "episode_number", "episode_start", "episode_stop", "historical_flag")`
- For each, compute `setdiff(expected, colnames(df))`. If any missing, `stop(glue("ERROR: RDS missing expected columns: {paste(missing, collapse=', ')}"))`.
- Log: `message("  Column validation passed")`

**SECTION 4: SELECT AND ORDER COLUMNS (per D-01)**
- Episodes (bars table): `episodes_export <- episodes %>% select(patient_id, treatment_type, episode_number, episode_start, episode_stop, episode_length_days, distinct_dates_in_episode, historical_flag, triggering_codes)`
- Detail (ticks table): `detail_export <- detail %>% select(patient_id, treatment_type, treatment_date, triggering_code, episode_number, episode_start, episode_stop, historical_flag)`

**SECTION 5: WRITE CSV OUTPUTS**
- `message("\n--- Writing CSV outputs ---")`
- `write.csv(episodes_export, OUTPUT_EPISODES, row.names = FALSE)` with message logging row count
- `write.csv(detail_export, OUTPUT_DETAIL, row.names = FALSE)` with message logging row count

**SECTION 6: FINAL SUMMARY**
- `message("\n=== Phase 01 Complete ===")`
- Log: unique patients in episodes, total episodes, total detail rows
- Log both output file paths
- Use `n_distinct(episodes_export$patient_id)` for patient count
- Use `format(nrow(...), big.mark = ",")` for comma-formatted counts (per Phase 45 lesson: R uses `format()`, not Python glue `:,` spec)
