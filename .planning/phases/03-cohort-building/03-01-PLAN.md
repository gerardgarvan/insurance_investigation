**Function 1: has_hodgkin_diagnosis() -- per D-01 (first filter)**

Tibble-in/tibble-out. Finds all patient IDs with at least one HL diagnosis in pcornet$DIAGNOSIS using is_hl_diagnosis(DX, DX_TYPE) from utils_icd.R. Returns only patients from input tibble that have HL diagnosis. Uses semi_join (not inner_join) to preserve input tibble structure without adding columns. This handles CHRT-03 (ICD format matching) because is_hl_diagnosis() internally calls normalize_icd() which strips dots for both input codes and reference codes.

```r
has_hodgkin_diagnosis <- function(patient_df) {
  hl_patients <- pcornet$DIAGNOSIS %>%
    filter(is_hl_diagnosis(DX, DX_TYPE)) %>%
    distinct(ID)

  message(glue("[Predicate] has_hodgkin_diagnosis: {nrow(hl_patients)} patients with HL diagnosis in DIAGNOSIS table"))

  patient_df %>%
    semi_join(hl_patients, by = "ID")
}
```

Tibble-in/tibble-out. Gets distinct patient IDs from pcornet$ENROLLMENT. Returns only patients from input that have at least one enrollment record. Per D-03: no minimum duration check. Per D-12: patients with HL diagnosis but no enrollment record are excluded.

**Function 3: exclude_missing_payer() -- per D-04**

Tibble-in/tibble-out. Accepts payer_summary as second argument. Filters to patients where PAYER_CATEGORY_PRIMARY is NOT NA, NOT "Unknown", NOT "Unavailable". Uses semi_join against the filtered payer_summary to retain only patients with concrete payer categories (Medicare, Medicaid, Dual eligible, Private, Other government, No payment / Self-pay, Other).

```r
exclude_missing_payer <- function(patient_df, payer_summary) {
  valid_payer_patients <- payer_summary %>%
    filter(
      !is.na(PAYER_CATEGORY_PRIMARY) &
      !PAYER_CATEGORY_PRIMARY %in% c("Unknown", "Unavailable")
    ) %>%
    distinct(ID)

  message(glue("[Predicate] exclude_missing_payer: {nrow(valid_payer_patients)} patients with valid payer category"))

  patient_df %>%
    semi_join(valid_payer_patients, by = "ID")
}
```

**Function 4: has_chemo() -- per D-05, D-06**

Returns a tibble of patient IDs with chemotherapy evidence. Combines:
- TUMOR_REGISTRY1: checks if CHEMO_START_DATE_SUMMARY column exists, filters where non-NA
- TUMOR_REGISTRY2/TR3: checks if DT_CHEMO column exists, filters where non-NA
- PROCEDURES: filters where PX_TYPE == "CH" and PX %in% TREATMENT_CODES$chemo_hcpcs
- PRESCRIBING: filters where RXNORM_CUI %in% TREATMENT_CODES$chemo_rxnorm

CRITICAL: Use conditional column checks: `if ("DT_CHEMO" %in% names(pcornet$TUMOR_REGISTRY2))` because TR1 has CHEMO_START_DATE_SUMMARY instead of DT_CHEMO (Pitfall 3 from research).

Returns tibble with columns ID, HAD_CHEMO (integer 1 for all rows -- these are patients WITH evidence). Caller will left_join and replace_na(0L) for patients without evidence.

```r
has_chemo <- function() {
  chemo_ids <- character(0)

  # TUMOR_REGISTRY1: CHEMO_START_DATE_SUMMARY (different column name from TR2/TR3)
  if (!is.null(pcornet$TUMOR_REGISTRY1) &&
      "CHEMO_START_DATE_SUMMARY" %in% names(pcornet$TUMOR_REGISTRY1)) {
    tr1_chemo <- pcornet$TUMOR_REGISTRY1 %>%
      filter(!is.na(CHEMO_START_DATE_SUMMARY)) %>%
      pull(ID)
    chemo_ids <- c(chemo_ids, tr1_chemo)
  }

  # TUMOR_REGISTRY2: DT_CHEMO
  if (!is.null(pcornet$TUMOR_REGISTRY2) &&
      "DT_CHEMO" %in% names(pcornet$TUMOR_REGISTRY2)) {
    tr2_chemo <- pcornet$TUMOR_REGISTRY2 %>%
      filter(!is.na(DT_CHEMO)) %>%
      pull(ID)
    chemo_ids <- c(chemo_ids, tr2_chemo)
  }

  # TUMOR_REGISTRY3: DT_CHEMO
  if (!is.null(pcornet$TUMOR_REGISTRY3) &&
      "DT_CHEMO" %in% names(pcornet$TUMOR_REGISTRY3)) {
    tr3_chemo <- pcornet$TUMOR_REGISTRY3 %>%
      filter(!is.na(DT_CHEMO)) %>%
      pull(ID)
    chemo_ids <- c(chemo_ids, tr3_chemo)
  }

  # PROCEDURES: chemo HCPCS codes
  if (!is.null(pcornet$PROCEDURES)) {
    px_chemo <- pcornet$PROCEDURES %>%
      filter(PX_TYPE == "CH" & PX %in% TREATMENT_CODES$chemo_hcpcs) %>%
      pull(ID)
    chemo_ids <- c(chemo_ids, px_chemo)
  }

  # PRESCRIBING: chemo RXNORM codes
  if (!is.null(pcornet$PRESCRIBING)) {
    rx_chemo <- pcornet$PRESCRIBING %>%
      filter(!is.na(RXNORM_CUI) & RXNORM_CUI %in% TREATMENT_CODES$chemo_rxnorm) %>%
      pull(ID)
    chemo_ids <- c(chemo_ids, rx_chemo)
  }

  result <- tibble(ID = unique(chemo_ids), HAD_CHEMO = 1L)
  message(glue("[Treatment] has_chemo: {nrow(result)} patients with chemotherapy evidence"))
  result
}
```

**Function 6: has_sct() -- per D-05, D-06, D-07**

Combines:
- TUMOR_REGISTRY2/TR3: DT_HTE column (hematologic transplant/endocrine therapy date). NOTE: DT_HTE may include endocrine therapy, not just SCT -- but for HL, endocrine therapy is not standard, so DT_HTE evidence in an HL cohort is reasonable SCT signal.
- TUMOR_REGISTRY1: HEMATOLOGIC_TRANSPLANT_AND_ENDOC column (non-date code field -- check if non-empty/non-zero as evidence)
- PROCEDURES: PX_TYPE == "CH" and PX %in% TREATMENT_CODES$sct_cpt

Returns tibble with ID, HAD_SCT (integer 1). Per D-07: single flag, no auto/allo distinction.

has_sct <- function() {
  sct_ids <- character(0)

  # TUMOR_REGISTRY1: HEMATOLOGIC_TRANSPLANT_AND_ENDOC (code field, not date)
  if (!is.null(pcornet$TUMOR_REGISTRY1) &&
      "HEMATOLOGIC_TRANSPLANT_AND_ENDOC" %in% names(pcornet$TUMOR_REGISTRY1)) {
    tr1_sct <- pcornet$TUMOR_REGISTRY1 %>%
      filter(!is.na(HEMATOLOGIC_TRANSPLANT_AND_ENDOC) &
             HEMATOLOGIC_TRANSPLANT_AND_ENDOC != "" &
             HEMATOLOGIC_TRANSPLANT_AND_ENDOC != "00") %>%
      pull(ID)
    sct_ids <- c(sct_ids, tr1_sct)
  }

      "DT_HTE" %in% names(pcornet$TUMOR_REGISTRY2)) {

      filter(!is.na(DT_HTE)) %>%
      pull(ID)
    sct_ids <- c(sct_ids, tr2_sct)

      "DT_HTE" %in% names(pcornet$TUMOR_REGISTRY3)) {

      filter(!is.na(DT_HTE)) %>%
