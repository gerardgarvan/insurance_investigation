<objective>
Enrich both Gantt chart CSV exports with human-readable code descriptions so third-party consumers can understand what each triggering code means without external lookups.

Purpose: The current gantt_detail.csv and gantt_episodes.csv contain bare treatment codes (J9000, 77427, etc.) that require domain expertise or external lookup to interpret. Adding descriptions makes the CSVs self-documenting for any downstream consumer.

Output:
- R/48_build_code_descriptions.R (new helper script)
- cache/outputs/code_descriptions.rds (new artifact: named character vector)
- R/49_gantt_data_export.R (modified: loads lookup, adds description columns)
- output/gantt_detail.csv (modified: gains triggering_code_description column)
- output/gantt_episodes.csv (modified: gains triggering_code_descriptions column)
</objective>

Create R/48_build_code_descriptions.R that builds a named character vector (code -> description) from 4 sources and saves as code_descriptions.rds. Per D-01 (static lookup, no API calls) and D-02 (standalone RDS file, named character vector).

**Script structure (follow existing R/NN_*.R conventions):**

HEADER COMMENT BLOCK:
```
# =============================================================================
# Phase 02: Build Code Description Lookup
# =============================================================================
# Creates a static named character vector mapping treatment codes to human-readable
# descriptions. Combined from 4 sources in precedence order (last wins):
#   1. Phase 39 RDS: unmatched_codes_classified.rds (CPT/HCPCS NLM API results)
#   2. Phase 40 RDS: unmatched_ndc_classified.rds (NDC/RXNORM RxNorm API results)
#   3. R/45 hardcoded_descriptions: retired radiation CPT codes (31 entries)
#   4. R/00_config.R inline comments: original curated codes with human descriptions
#
# Decision traceability:
#   D-01: Static lookup from existing sources, no NLM API at runtime
#   D-02: Output as code_descriptions.rds named character vector
#   D-05: Missing descriptions handled downstream (empty string)
#
# Inputs:
#   - cache/outputs/unmatched_codes_classified.rds (Phase 39)
#   - cache/outputs/unmatched_ndc_classified.rds (Phase 40)
#   - R/45_radiation_cpt_audit.R hardcoded_descriptions (sourced inline)
#   - Manual transcription from R/00_config.R inline comments
#
# Output:
#   - cache/outputs/code_descriptions.rds (named character vector)
# =============================================================================
```
