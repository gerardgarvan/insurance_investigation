    OPTIMIZE 17_value_audit.R:

    The HIPAA suppression logic is duplicated: once in the main audit loop (lines 239-253) and once in the derived audit loop (lines 311-325). Both blocks are identical.

    Extract into a helper function named apply_hipaa_to_audit():
    ```r
    #' Apply HIPAA suppression to audit result tibble
    #'
    #' Replaces n counts 1-10 with "<11" and marks their pct as "suppressed"
    #'
    #' @param result Audit result tibble with n and pct columns
    #' @return Tibble with HIPAA-suppressed n and pct
    apply_hipaa_to_audit <- function(result) {
      result %>%
        mutate(
          n_raw = suppressWarnings(as.numeric(n)),
          n = case_when(
            is.na(n_raw) ~ as.character(n),
            n_raw >= 1 & n_raw <= 10 ~ "<11",
            n_raw == 0 ~ "0",
            TRUE ~ format(n_raw, big.mark = ",", trim = TRUE)
          ),
          pct = case_when(
            n == "<11" ~ "suppressed",
            TRUE ~ as.character(pct)
          )
        ) %>%
        select(-n_raw)
    }
    ```
    Place this helper after the existing suppress_hipaa() function (around line 57).
    Then replace both duplicate blocks with `result <- apply_hipaa_to_audit(result)`.

    Per D-06: Style-preserving only.
    Per D-07: No function signature changes on public functions.
