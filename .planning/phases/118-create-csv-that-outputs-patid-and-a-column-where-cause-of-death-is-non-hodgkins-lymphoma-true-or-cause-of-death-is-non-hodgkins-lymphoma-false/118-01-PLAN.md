<objective>
Produce a NEW standalone "100+" investigation script that writes a CSV with one row per
DECEASED patient carrying `PATID` and a THREE-STATE column `cause_of_death_is_nhl` indicating
whether that patient's cause of death classifies as Non-Hodgkin Lymphoma (TRUE / FALSE / blank).

Purpose: Give the team a simple, joinable per-patient flag distinguishing NHL cause-of-death
from other coded causes and from uncoded/missing causes — the blank state matters because
DEATH_CAUSE is sparsely coded and collapsing "missing" into FALSE would misrepresent the data
(CONTEXT.md D-04).

Output:
  - output/death_cause_nhl_flag.csv (PATID, cause_of_death_is_nhl)
  - R/102_death_cause_nhl_flag.R (new standalone script)

DEATH-table derivation (copy exactly from R/35 SECTION 2, lines ~62-117):
```r
death_raw <- get_pcornet_table("DEATH") %>% collect()

# Field-availability guard (D-78-01)
death_cause_available <- FALSE
death_cause_col <- NULL
if ("DEATH_CAUSE" %in% names(death_raw)) {
  death_cause_col <- "DEATH_CAUSE"; death_cause_available <- TRUE
} else if ("DEATH_CAUSE_CODE" %in% names(death_raw)) {
  death_cause_col <- "DEATH_CAUSE_CODE"; death_cause_available <- TRUE
} else {
  message("  WARNING: DEATH_CAUSE field not available in DEATH table")
}

death_data <- death_raw %>%
  mutate(DEATH_DATE = parse_pcornet_date(DEATH_DATE)) %>%
  mutate(DEATH_DATE = if_else(year(DEATH_DATE) == 1900L, as.Date(NA), DEATH_DATE)) %>%
  filter(!is.na(DEATH_DATE))

if (death_cause_available) {
  death_data <- death_data %>% select(ID, DEATH_DATE, DEATH_CAUSE = all_of(death_cause_col))
} else {
  death_data <- death_data %>% mutate(DEATH_CAUSE = NA_character_) %>% select(ID, DEATH_DATE, DEATH_CAUSE)
}

death_data <- death_data %>%
  group_by(ID) %>%
  summarise(DEATH_DATE = min(DEATH_DATE), DEATH_CAUSE = first(DEATH_CAUSE), .groups = "drop")
```
