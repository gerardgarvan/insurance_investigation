## Audit Results: Function Signatures and Differences

### 1. is_missing_payer() -- 4 copies

**Copy A (R/19_flm_duplicate_dates.R:31)** -- OLDER version:
```r
is_missing_payer <- function(payer_value) {
  is.na(payer_value) |
    nchar(trimws(payer_value)) == 0 |       # <-- uses nchar(trimws())
    payer_value %in% c("NI", "UN", "OT", "99", "9999")
}
```

**Copy B (R/21_all_site_duplicate_dates.R:57, R/23_overlap_classification.R:57, R/34_overlap_classification_av_th.R:58)** -- CURRENT version (3 identical copies):
```r
is_missing_payer <- function(payer_value) {
  is.na(payer_value) |
    payer_value == "" |                       # <-- uses direct "" check (Phase 32 DuckDB fix)
    payer_value %in% c("NI", "UN", "OT", "99", "9999")
}
```

**Difference:** Copy A uses `nchar(trimws(payer_value)) == 0`, Copies B use `payer_value == ""`.
The `== ""` version was intentionally updated in Phase 32 to fix DuckDB translation gap #7.
**Decision: Use Copy B (the `== ""` version) as canonical.** Copy A in R/19 is stale.

### 2. CODE_TO_TIER() -- 3 copies

**All 3 copies are functionally identical** (R/36:88, R/45:60, R/46:69).
Only whitespace alignment differs (extra space after "Medicaid" in R/45 and R/46).
All map 8 AMC payer categories to 8 tiers via case_when(), with NA and TRUE fallback to "Missing".
**Decision: Use any copy as canonical (they are identical).**

### 3. style_table() -- 2 copies with MEANINGFUL DIFFERENCES

**Copy A (R/11_generate_pptx.R:684):**
```r
style_table <- function(ft, total_row = integer(0))
```
- body font size hardcoded to 12, header to 13
- `bold(j = 1, part = "body")` -- bolds first column
- padding: 6px left/right, 3px top/bottom
- `odd_rows <- seq(1, n_rows, by = 2)` -- no empty-guard

**Copy B (R/22_generate_phase19_20_pptx.R:59):**
```r
style_table <- function(ft, total_row = integer(0), body_font_size = 11)
```
- body font size parameterized (default 11), header is 12
- NO `bold(j = 1)` -- first column not bolded
- padding: 5px left/right, 2px top/bottom
- `odd_rows <- if (n_rows >= 1) seq(...) else integer(0)` -- has empty-guard
- add_table_slide() also defined in R/22, using style_table with body_font_size

**Decision: Create a SUPERSET version** with `body_font_size` parameter (default 12 to match
R/11's existing behavior), `header_font_size` parameter (default 13), `bold_first_col` parameter
(default TRUE to match R/11), `padding` parameter (default 6 to match R/11). R/22 will call
with `body_font_size = 11, header_font_size = 12, bold_first_col = FALSE, padding = 5`.
Include the empty-guard from Copy B (strictly better).

### 4. field_match() -- 2 copies (IDENTICAL)

Both in R/23_overlap_classification.R:64 and R/34_overlap_classification_av_th.R:65.
Exact same body: both_na, one_na, return logic. **Use either as canonical.**

### 5. check_file() -- 2 copies (IDENTICAL)

Both in R/43_test_durations.R:283 and R/44_test_episodes.R:281.
Exact same body: file.exists check, size formatting, message output. **Use either as canonical.**
